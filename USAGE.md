# OpenClaw 人设编辑器 - 使用说明

## 🎭 重要特性

### ✅ 智能更新
- **只更新相关字段** — 身份信息（名字、性格、世界观）
- **保留自定义内容** — 你的喜好、秘密基地等完全保留
- **智能合并** — 不是覆盖，而是精确编辑

### ✅ 安全可靠
- **先请求权限** — 明确告知将要修改什么
- **预览更改** - 显示所有将要进行的修改
- **确认机制** — 你确认后才会真正修改

### ✅ 灵活通用
- **支持任意类型** — 动物、机器人、角色、概念...
- **完全可定制** — 所有选项都可以自由编辑
- **多次使用** — 可以反复优化人设

## 🚀 使用方法

### 方法一：命令行运行

```bash
cd /root/.openclaw/workspace/skills/openclaw-persona-editor
python3 scripts/create_persona.py
```

按照提示一步步回答问题：
1. 权限确认
2. 选择人设类型
3. 填写各个维度
4. 预览更改
5. 确认应用

### 方法二：在对话中触发

对 OpenClaw 助手说：
- "创建 AI 助手身份"
- "定义新的人设"
- "修改我的助手性格"
- "更新身份设定"

助手会自动触发此技能。

## 📝 更新逻辑说明

### IDENTITY.md 更新规则

**只更新这些字段：**
- Name（名字）
- Nickname（昵称）
- Gender（性别）
- Creature（物种/类型）
- Origin（来源）
- Age（年龄描述）
- Vibe（整体氛围）
- Emoji（专属符号）

**保留：**
- 你添加的所有其他部分（如"我的喜好"、"秘密基地"等）

### SOUL.md 更新规则

**只更新这些部分：**
- 🧠 核心世界观
- Core Truths（核心特质、行为模式、反差萌点）
- 🎨 语言风格

**保留：**
- 你自定义的其他章节
- 你的 Boundaries、Continuity 等自定义内容

### USER.md 更新规则

**只在文件为空时**添加基本模板，否则保留你的所有内容。

### MEMORY.md 更新规则

**追加**本次更新记录，不修改原有内容。

## ⚠️ 注意事项

1. **备份重要** — 虽然工具很智能，但建议首次使用前备份 workspace
2. **仔细预览** — 应用前仔细检查预览，确认无误再继续
3. **逐步优化** — 可以多次使用，每次优化一个维度
4. **保留自定义** — 你的所有自定义内容都会被保留

## 🔄 工作流程

```
1. 请求权限
   ↓
2. 收集信息（6 个维度）
   ↓
3. 读取现有文件
   ↓
4. 智能合并（保留自定义内容）
   ↓
5. 预览更改
   ↓
6. 确认应用
   ↓
7. 完成
```

## 💡 常见问题

**Q: 会删除我的自定义内容吗？**
A: 不会！工具只会更新身份相关字段，你的所有自定义内容（如喜好、笔记等）都会完整保留。

**Q: 可以多次使用吗？**
A: 可以！每次运行都是智能合并，你可以反复优化人设。

**Q: 如果不满意怎么办？**
A: 预览阶段会显示所有更改，你可以选择不应用。如果已经应用，可以使用 git 恢复（如果有版本控制）。

**Q: 支持哪些人设类型？**
A: 任意类型！动物、机器人、角色、概念...没有任何限制。
